# Number Guessing Game 🎯

A fun terminal-based Number Guessing Game built with Python where 
you try to guess a randomly generated number between 1 and 100!

## Features
- Randomly generates a number between 1 and 100
- Tells you if your guess is too high or too low
- Congratulates you when you guess correctly
- Keeps going until you find the correct number

## How to Run
1. Make sure Python is installed on your computer
2. Run the script:
   python number_guessing_game.py
3. Enter a number between 1 and 100
4. Follow the hints:
   - "Your prediction is low"  = guess higher
   - "Your prediction is high" = guess lower
5. Keep guessing until you win!

## Technologies Used
- Python 3
- Random module
